# Імітаційний прототип системи продажу нерухомості
Проєкт демонструє взаємодію між покупцем, продавцем, агентом з нерухомості та банком при здійсненні операцій з нерухомістю.

У цій версії всі .java файли були перероблені на Python (.py) зі збереженням класів та логіки взаємодії.

## Компіляція
cd src
python -m py_compile *.py

## Запуск
cd src
python Main.py

## Структура проєкту
- src/ - каталог з вихідним кодом
  - Reviewable.py - інтерфейс (ABC) для перегляду пропозицій
  - Participant.py - абстрактний клас учасника
  - Deal.py - клас угоди
  - Buyer.py - клас покупця
  - Seller.py - клас продавця
  - Agent.py - клас агента
  - Bank.py - клас банку
  - Main.py - головний файл з демонстрацією
- docs/ - документація проєкту
  - requirements.md - вимоги до програми
  - usecase.puml - діаграма варіантів використання
  - class.puml - діаграма класів
  - sequence.puml - діаграма послідовності
  - state.puml - діаграма станів
